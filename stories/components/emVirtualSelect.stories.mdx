# select-v
传递`value-key`时，绑定的是一个`sring[]`或者`number[]`, 不传`value-key`时，绑定的是一个`object[]`,此时必须设置`unique-key`以区分唯一键名
因为该组件主要处理大数据量的情况，默认单个item是一个对象，如果非要处理[1,2,3,4...]这样的数据要在传入的时候转一下结构
